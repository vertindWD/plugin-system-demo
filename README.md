# Go Plugin Execution System

这是一个轻量级、高可用、基于 Go 原生 `plugin` 模块实现的动态插件执行引擎。
本项目旨在展示基于 Go 语言的接口解耦设计以及后端高可用组件的防崩溃与超时治理方案。

## ✨ 核心特性

- **动态化加载**：使用 Go 标准库 `plugin` 实现在运行时动态加载 `.so` 文件，无需重新编译主程序。
- **契约式设计**：通过 `core.Plugin` 接口统一规范插件的输入输出 (`map[string]interface{}`)，主程序与业务插件完全解耦。
- **异常崩溃隔离**：管理器内部捕获插件执行过程中的致命错误，防止单个劣化插件导致宿主进程崩溃。
- **执行超时控制**：基于 `context.Context` 实现插件执行时间的严格控制，避免流水线因第三方逻辑阻塞而被拖垮。
